/// Description
  
Three simple C# scripts used to perform basic logical functions by converting up to two inputs, into a single output.

The script allows users to choose the logic gate type (AND, OR, XOR, NOT, NAND, NOR and XNOR) from the inspector in Unity.


/// Important

The scripts require use of the Gaskellgames shared library scripts for custom inspector properties: Inspector Events & Readonly Inspector Fields.
